# Manual QA Project

## Project Overview
This repository contains an end-to-end Manual QA project for testing the Login module of the Drishti IAS learning platform. The focus is on validating UI, functional behavior, error handling, CAPTCHA validation, and API responses.

## Tested URL
https://www.drishtilearningapp.com/

## Scope
- Login UI validation
- Field validation (Email, Password)
- CAPTCHA validation
- Remember Me functionality
- Error message handling
- API validation (Login endpoint)

## Tools Used
- Excel / Google Sheets
- Postman
- GitHub
